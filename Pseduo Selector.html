Pseudo Class

CSS pseudo classes are used to add special effects to some selector. It doessnot required any JavaScript knowledge
to use those effects.

:link
This class is Use to add special style to an unvisted Link.
:visited
This class is use to add special style to a visied link.
:hover
This class is use to add special style to an element when you mouse over it.
:active
This class is use to add special style to an active element.
:focus
This class is use to add special style to an element while the element has focus
:nth child(__)
This Class is use to add special style to the nth child(_) of spcific section.
