Type of selector in CSS

Element with class Selector

The element .class selector is used to select the specified element with the the specified class.

CSS Syntax Element with class Selector
element .class{
css Declaration;
}

Child Selector

The child combinator (>)is placed between two CSS Selectors,It matches only those element matched by
the second selector that are the direct children of elements matched by the first.Elements matched by
the second selectors must be the immediate children of the elements matched by the first selector.

CSS Syntax for Child Selector
selector1 > Selector2{
style properties;
}
example
div > span {
background-color: Green;
}

Descendent Selector

The descendant selector matches all elements that are descending that are descendants of a specified
element i.e all that specified element present inside the selector1 get update.

CSS Syntax for Descendent Selector

selector1 Selector 2 {
CSS Declaration
}
Example
div p { color:Green; font-size:20px; }
